
# BlackJack

Консольная реализация популярной карточной игры «Блэкджек» на C++ с визуализацией карт в ASCII-графике, поддержкой страховки, удвоения ставки и других классических механик.

## Правила игры

Игра ведётся по стандартным правилам Блэкджека:

- Игрок делает ставку, не превышающую его банк.
- Дилер и игрок получают по две карты, одна карта дилера — закрытая.
- Игрок может **брать** карты (Hit) или **остановиться** (Stand).
- Номиналы карт: 2–10 — соответствующее количество очков; валет, дама, король — 10; туз — 11 (или 1 при переборе).
- Если сумма очков превышает 21 — игрок проигрывает.
- После игрока ходит дилер: берёт карты, пока сумма меньше 17.
- Победитель определяется по большей сумме очков, не превышающей 21; при равенстве — ничья.

**Дополнительные возможности:**
- **Double Down** — удвоение ставки с обязательным добором ровно одной карты.
- **Insurance** — страховка при тузе у дилера (дополнительная ставка 0.5 от начальной).
- Выигрыш с блэкджеком (две карты = 21) оплачивается 3:2.
- Автоматический выигрыш при двух тузах у игрока против руки дилера ≤20.

Подробные правила — в файле [`rules.txt`](rules.txt).

## Особенности реализации

- Полноценная консольная ASCII-графика карт (масти и достоинства отображаются псевдографикой).
- Поддержка всех основных действий: Hit, Stand, Double, Insurance.
- Корректная обработка мягких рук (два туза у игрока/дилера).
- Система банка: начальный баланс, ставки, возврат при ничьей.
- Очистка консоли между раундами (под Windows и Unix-системами).
- Простой текстовый ввод команд с проверкой корректности.

## Требования

- Компилятор с поддержкой **C++17**.
- **CMake** версии 3.10.2 или выше.
- Операционная система: Windows, Linux, macOS (используется `system("cls")` / `system("clear")`).

## Сборка и запуск

```bash
# Клонирование репозитория
git clone <url>
cd BlackJack

# Сборка
cmake -B build
cmake --build build

# Запуск
./build/BlackJack
```
На Windows для генерации проекта можно использовать, например:

```bash
cmake -B build -G "Visual Studio 17 2022"
cmake --build build --config Release
```
Управление
Игра управляется текстовыми командами. Допустимые варианты ответа (регистр важен, но поддерживаются все комбинации):

Действие	Команды для ввода
Начать игру / согласиться	YES, Yes, yes
Отказаться	NO, No, no
Взять карту	HIT, Hit, hit
Остановиться	STAND, Stand, stand
Удвоить ставку	DOUBLE, Double, double
Страховка (да)	YES, Yes, yes
Страховка (нет)	NO, No, no
При вводе ставки необходимо указать положительное целое число, не превышающее текущий банк.
